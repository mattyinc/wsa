# Prompt to Give Codex

Read the repository instructions before coding:

1. `AGENTS.md`
2. `README.md`
3. `ROADMAP.md`
4. Everything under `/docs`

Build the Wall Street Academy MVP app.

Important:
- Markdown is the source of truth.
- Do not hardcode curriculum content into React components.
- Render content from `/docs` and `/curriculum`.
- Build a white-first app with a dark mode switch.
- Prioritize the lesson reading experience.
- Avoid generic AI website design.
- Avoid excessive gradients, glassmorphism, stock photos, hype language, and flashy animations.

Use:
- Next.js
- TypeScript
- Tailwind CSS
- Markdown or MDX rendering
- LocalStorage for progress
- LocalStorage for Analyst Notebook

Build only the MVP:
- Home page
- Course dashboard
- Lesson reader
- Notebook
- About page
- Theme toggle
- Markdown rendering
- Local progress tracking

Do not build:
- Auth
- Database
- Payment
- AI chat
- Admin panel
- Real market data

Start by creating the app shell and a polished lesson reader.
