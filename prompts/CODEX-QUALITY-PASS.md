# Prompt to Give Codex After First Build

Review the current WSA app for quality.

Focus on:

- readability
- spacing
- typography
- light mode polish
- dark mode polish
- lesson reader experience
- markdown rendering
- localStorage behavior
- responsive layout

Fix anything that feels generic, cramped, flashy, or hard to read.

The app should feel like a premium learning platform, not a SaaS landing page.

Do not add new major features.

Improve the quality of the existing MVP.
