# WSA Project Status

## Current Phase

MVP complete.

## Current Goal

Expand the Course 01 curriculum while preserving the polished reading experience.

## Current Product Decision

The app should support dark mode, but light mode is the default.

## Source of Truth

Markdown files in:

- `/docs`
- `/curriculum`

## Current MVP Scope

Build:

- Home page
- Course dashboard
- Lesson reader
- Analyst Notebook using localStorage
- Progress tracking using localStorage
- Theme toggle
- Markdown rendering

Do not build:

- Authentication
- Payment system
- Backend database
- Real market data integration
- AI mentor chat
- User accounts
- Admin panel

Those come later.

## Current Design Direction

White-first.

Calm.

Editorial.

Premium learning.

No AI slop.

No hype.

No generic SaaS look.

## Completed Milestone

A polished, responsive lesson reader using Markdown as source content, with local progress,
notebook entries, and light/dark themes.
